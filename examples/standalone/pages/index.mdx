# Welcome to My Documentation

This is a Nextra documentation site built with Bazel rules.

## Features

- **Static Site Generation**: Complete static documentation site
- **MDX Support**: Write documentation in MDX format
- **Bazel Integration**: Seamless integration with Bazel build system
- **Hot Reload**: Development mode with hot reload support

## Getting Started

1. Add the Nextra rules to your BUILD file
2. Create your documentation pages in MDX format
3. Build and deploy your documentation site
